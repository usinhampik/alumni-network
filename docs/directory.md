# Alumni Directory

Welcome to the Alumni Directory!

## Overview
This document serves as a guide for maintaining the alumni directory, including how to add new profiles.

## Alumni Directory Information
- **Name:** The full name of the alumnus/alumna.
- **Graduation Year:** The year of graduation.
- **Current Position:** The current job title.
- **Company:** The name of the organization the alumnus/alumna is currently employed with.
- **Contact Information:** Include email or any other relevant contact info (if public).

## How to Add Profiles
1. **Open the Alumni Directory file:** Navigate to this file in the repository.
2. **Add a new entry:** Follow the format provided in the document to ensure consistency.
3. **Commit your changes:** Use clear commit messages to describe the modifications.
4. **Pull requests:** If you're collaborating, submit a pull request for review.

Please ensure that you have permission to share any contact information.
